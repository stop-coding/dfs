#                    build
*【说明】编译工程采用cmake编译

* for linux (ubuntu)
* apt-get install cmake

	1. cd code root;
	2. mkdir build
	3. cd build
	4. cmake ../    (default:debug mode)
	5. make -j
	6. make install
